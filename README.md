# 👵 OldFit - Seguridad y Bienestar

[![Build Status](https://img.shields.io/badge/status-active-brightgreen)](https://tu-link.com)
[![React Native](https://img.shields.io/badge/React_Native-v0.7x-blue)](https://reactnative.dev/)

OldFit es una plataforma integral que combina el rastreo GPS de alta precisión con ejercicios de estimulación mental para adultos mayores.

## 🚀 Instalación rápida

1. **Clonar:** `git clone https://github.com/checo-coder/Gariatria_movil.git`
2. **Dependencias:** `npm install`
3. **Env:** Configura tu `EXPO_PUBLIC_API_URL` en un archivo `.env`.
4. **Correr:** `npx expo start`

## 🛡️ Permisos y Seguridad
- **Location (Always):** Permite el rastreo mediante Sockets incluso en segundo plano.
- **Geocencing:** Alertas automáticas al servidor cuando el usuario sale del perímetro definido.
